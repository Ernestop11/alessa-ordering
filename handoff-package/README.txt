====================================
LAS REINAS - PRINT RELAY SETUP
====================================

WHAT THIS DOES:
- Connects to your thermal printer automatically
- Prints orders from the website when you click "Print" in fulfillment
- Works on any WiFi network (auto-discovers printer)

REQUIREMENTS:
1. Node.js installed (https://nodejs.org)
2. Thermal printer connected to same WiFi as this Mac
3. Printer has DHCP enabled (gets IP automatically)

HOW TO START:
1. Double-click "start-print-relay.command"
2. If prompted, allow it to run (System Preferences > Security)
3. Wait for "Found printer at X.X.X.X" message
4. Keep the Terminal window open while using

TROUBLESHOOTING:
- "No printer found" = Check printer is on same WiFi
- "Connection timeout" = Printer may be asleep, try again
- Window closes immediately = Node.js not installed

# Taqueria Rosita Assets

## Required Images

Place the following images in this directory:

- `logo.png` - Logo (recommended: 200x200px, transparent PNG)
- `hero.jpg` - Hero banner (recommended: 1920x600px, JPG)
- `membership.jpg` - Membership/rewards background (recommended: 1200x400px, JPG)

## Optional Images

- `images/menu-items/*.jpg` - Menu item photos (recommended: 800x600px, JPG)

## Usage

These assets are served at:
- Logo: `/tenant/taqueriarosita/logo.png`
- Hero: `/tenant/taqueriarosita/hero.jpg`
- Membership: `/tenant/taqueriarosita/membership.jpg`
- Menu items: `/tenant/taqueriarosita/images/menu-items/{filename}.jpg`

## Brand Colors

- Primary: #dc2626 (red-600) - Classic taqueria red
- Secondary: #fbbf24 (amber-400) - Warm accent
- Theme: #7f1d1d (red-900) - Status bar

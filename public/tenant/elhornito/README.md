# El Hornito Bakery Assets

This directory contains assets for El Hornito Bakery, a sub-tenant of La Poblanita.

## Required Assets

1. **logo.png** - El Hornito bakery logo (256x256 PNG recommended)
2. **hero.jpg** - Hero banner image for the bakery page (1600x900 recommended)

## Theme Colors

- Primary: #d97706 (amber-600)
- Secondary: #f59e0b (amber-500)
- Theme/Status bar: #78350f (amber-900)

## Notes

- El Hornito shares cart with La Poblanita (customers can order from both)
- Menu is managed via the La Poblanita admin under the "Panaderia" tab
- The bakery page is accessible at /bakery when visiting lapoblanita domain
